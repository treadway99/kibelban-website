# Kibelban 2027

Official campaign website for Kandagor James Kipkong'a "Kibelban", UDA candidate for Baringo County Governor 2027.

## Overview

This is a single-page campaign site built with plain HTML and CSS. It introduces the candidate, highlights the core campaign agenda, and includes a photo gallery plus an embedded video section.

## Features

- Hero section with campaign branding
- Candidate background and short bio section
- Five campaign pillars under the Soko la Ushindi agenda
- Photo gallery with lightbox viewing
- Embedded YouTube preview for campaign updates

## Project Files

- `index.html` - main page markup and lightbox script
- `styles.css` - site styling and layout
- `Kibelban.jpg` - hero portrait
- `Feeding1.png`, `Feeding2.png`, `PeaceTalks.png` - gallery images

## Run Locally

1. Open `index.html` in a browser.
2. Or use a simple local server if you prefer.

Example with Python:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
